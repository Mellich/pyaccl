# ACCL Python Bindings

This package provides Python bindings for the [Alveo Collective Communication Library (ACCL)](https://github.com/Xilinx/ACCL). 

Built and uploaded with:
```sh
python3 -m build
python3 -m twine upload dist/*
```

Install as follows:
```sh
pip3 install pyaccl
```
