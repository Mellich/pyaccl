Welcome to PyACCL's documentation!
===================================

**PyACCL** is a `Pynq <https://github.com/Xilinx/PYNQ>`_-based Python driver for 
`ACCL <https://github.com/Xilinx/ACCL>`_ which enables MPI-like collective communication for Alveo FPGAs.
Check out the :doc:`usage` section first, and use our :doc:`api` reference to program with PyACCL.

.. note::

    This project is under active development.
